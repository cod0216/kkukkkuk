
# 🐾 KKUKKKUK - 반려동물 의료 기록 관리 서비스

## 🐶 서비스 소개

**KKUKKKUK**은 반려동물의 의료 기록을 안전하게 **보관하고 공유할 수 있는 서비스**입니다.  
보호자는 반려동물의 진료 기록을 앱을 통해 **편리하게 확인**할 수 있으며,  
병원은 보호자의 동의하에 해당 기록을 열람할 수 있습니다.


[👉 영상 포트폴리오 보러가기](https://drive.google.com/file/d/1X7G8SvDf38M0deGE9RCDhTUok_JvH6Mu/view?usp=sharing)

## 😺 서비스가 필요한 이유

- 병원을 변경할 때마다 **반려동물의 과거 진료 내용을 직접 설명해야 하는 불편함**이 존재합니다.  
- 보호자가 **예방접종 내역이나 병력 정보를 잊어버리면**, 정확한 진료가 어려워질 수 있습니다.  
- KKUKKKUK을 사용하면 **기록을 분실 걱정 없이 한 곳에 보관**하고, 필요 시에만 병원과 **안전하게 공유**할 수 있습니다.

## 🌐 웹(동물병원) 기능 소개

### 로그인
<img src="/image/웹_로그인.gif"/>

- MetaMask를 통한 **DID 인증**과 로그인 절차를 통해 서비스를 이용할 수 있습니다.  
- 개인 지갑인 **MetaMask**에 DID를 안전하게 저장하고, 인증 성공 시 로그인 완료됩니다.
- **MetaMask 인증과 로그인이 모두 성공**해야 꾹꾹 서비스를 이용할 수 있습니다

<br/>

---

### 진료 기록 조회
<img src="/image/진료_조회.gif"/>

- 보호자가 QR 코드를 찍어어 동물 병원에 **진료 정보 공유 동의(트랜잭션)** 를 하면,
- 병원은 블록체인 상의 **과거 진료 기록을 안전하게 조회**할 수 있습니다.
- 동물 병원은 진료 받을 동물의 과거 의료 기록부터 최신 정보까지 모두 조회하여, 더 정확한 진료가 가능합니다.

<br/>

---

### 진료 기록 작성
<img src="/image/진료_추가.gif"/>

- 병원은 반려동물을 진료하며 **진단명, 증상, 약물, 접종 내역** 등을 작성할 수 있습니다.
- 해당 의료 기록은 블록체인에 **트랜잭션을 통해 영구 저장**됩니다.
- 의료 기록은 중앙 서버를 거치지 않고, 블록체인에 안전하게 기록되게 됩니다.

<br/>

---


### 자동 완성
<img src="/image/자동_완성.gif"/>

- 병원이 진료 시 검사, 접종 정보를 조회할 때, 연관된 단어들이 드롭 다운 형태로 제공됩니다.
- 병원이 입력한 검사 및 접종 항목을 **Redis에 저장**합니다.
- 검색 시 자동 완성으로 **빠르고 정확한 입력**이 가능합니다.

<br/>

---


### 채팅 기능
<img src="/image/채팅.gif"/>

- 채팅 기능은 병원 간의 원활한 소통을 지원합니다.
- 다른 병원에서 진행된 진료 내용에 대해 추가 문의가 필요한 경우, 채팅으로 원활한 문의가 가능합니다.

<br/>

---

### 비인증 기록 인증
<img src="/image/인증.gif"/>

- 보호자가 진단서/영수증으로 직접 등록한 의료 기록의 경우 **병원에서 인증하여 공식 기록으로 전환**할 수 있습니다.
- 인증받은 병원에서는 해당 영수증 사진이나 검사, 접종, 약물 등의 내용을 확인한 후, 이를 **병원 인증 기록**으로 변경할 수 있습니다.

<br/>

## 📱 모바일(보호자 회원원) 기능 소개

### 카카오 로그인  
<img src="/image/로그인.gif" width="200"/>

- 카카오 계정으로 **간편 로그인**이 가능합니다.  
- 별도의 회원가입 없이 **소셜 로그인 기반으로 서비스를 이용할 수 있습니다.**

<br/>

---

### 니모닉 지갑 생성/복구  
<img src="/image/니모닉.gif" width="200"/>

- 니모닉 문구를 통해 **반려동물 지갑을 생성 또는 복구**할 수 있습니다.
- 니모닉 지갑은 **개인 키를 직접 보관**하는 방식으로, 반려동물의 정보를 안전하게 관리할 수 있습니다. 
- 지갑을 분실한 경우에도, 생성 시의 문구로 복구가 가능합니다. 

<br/>

---

### 반려동물 등록  
<img src="/image/반려동물등록.gif" width="200"/>

- 반려동물의 정보를 **디지털 지갑에 등록 및 관리**할 수 있습니다.  
- 등록된 동물에 대해서, **병원과 진료 기록을 공유**할 수 있습니다.

<br/>

---

### 스마트 컨트랙트  
<img src="/image/스마트컨트랙트.gif" style="width:200px"/> 

- 제휴 병원에 방문하여 **QR 코드**를 찍으면, **스마트 컨트랙트**를 할 수 있습니다. 
- 반려동물의 **진료 기록을 해당 병원에 공유**하고, 기록에 대한 권한을 줄 수 있습니다.
- 병원이 해당 반려동물에 대해 기록한 정보는 쉽게 조회할 수 있습니다.

<br/>

---


### 지갑 관리  
<img src="/image/지갑관리.gif" style="width:200px"/> 

- 니모닉 지갑으로 생성한 지갑을 관리할 수 있습니다. 
- 지갑은 **여러 사람이 공유가 가능**하기 때문에, 내 지갑, 임시보호용 지갑, 단체용 지갑 등 **용도에 맞춰서 관리가 가능**합니다.  
- **같은 지갑을 가진 보호자들의 목록도 조회**가 가능합니다.

<br/>

---

### 반려동물 의료기록 확인 
<img src="/image/반려동물의료기록조회.gif" style="width:200px"/> 

- 반려동물의 **지금까지의 의료 기록을 조회**할 수 있습니다.  
- **인증 마크**를 통해 병원이 입력한 정보 / 보호자가 직접 입력한 정보를 구분할 수 있습니다. 

<br/>

---

### 반려동물 의료기록 추가 
<img src="/image/ocr.gif" style="width:200px">/ 

- 다양한 형태의 반려동물 의료 기록을 **OCR 형태**로 추가가 가능합니다. 
- 보호자가 직접 입력한 기록은 **인증 마크가 표시되지 않습니다**.  
- 기록의 정확성을 위해 OCR 파싱된 데이터는 **AI로 가공**을 거치게 됩니다. 

<br/>

---

### 지도, 제휴 병원 확인 
<img src="/image/지도.gif" style="width:200px"/> 

- 사용자 **주변의 제휴 동물병원을 확인**할 수 있습니다.
- 제휴병원인 경우 **제휴 인증 마크**와 함께 표시됩니다.
